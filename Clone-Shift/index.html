Clone Shift

Clone이 스위치를 누르는 동안 문을 통과하세요.

Actions 0 / 14

Stage Rule

  • Player는 한 턴에 이동 또는 대기합니다.
  • Player 행동만 Actions를 1 소모합니다.
  • Clone 생성 시 이동 기록이 저장되고 Player는 시작점으로 돌아갑니다.
  • Clone은 저장된 이동 기록이 자동 재생합니다.
  • Clone이 스위치 위에 있으면 잠긴 문이 열립니다.
  • 스위치는 유지형과 고정형이 있습니다.
  • 유지형 스위치는 Player 또는 Clone이 밟고 있는 동안만 연결된 통로가 열립니다.
  • 고정형 스위치는 한 번 작동하면 연결된 통로가 계속 열립니다.
